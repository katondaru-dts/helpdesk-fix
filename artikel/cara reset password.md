# Cara Reset Password Akun di Aplikasi Helpdesk

Jika Anda lupa password atau tidak bisa login, ikuti panduan berikut sesuai metode login yang Anda gunakan.

## Login dengan Google (SSO)

Jika Anda login menggunakan akun Google (`@unmer.ac.id` atau `@student.unmer.ac.id`), aplikasi **tidak menggunakan password** — autentikasi sepenuhnya dikelola oleh Google.

Untuk reset password, lakukan melalui akun Google Anda:
1. Buka [myaccount.google.com](https://myaccount.google.com)
2. Pilih **Keamanan** → **Password**
3. Ikuti langkah yang diberikan Google

Setelah password Google diperbarui, login ke Helpdesk akan otomatis menggunakan password baru.

## Login dengan Email & Password Manual

Jika akun Anda dibuat secara manual oleh Admin (bukan via Google SSO):

### Langkah untuk Pengguna
Reset password **tidak bisa dilakukan sendiri**. Hubungi Administrator IT untuk meminta reset password dengan menyebutkan:
- Nama lengkap
- Email akun yang terdaftar

### Langkah untuk Administrator
1. Login ke aplikasi sebagai Admin
2. Buka menu **Admin** → **Manajemen User**
3. Cari nama atau email pengguna yang bersangkutan
4. Klik tombol **Edit**
5. Isi kolom **Password Baru** dengan password sementara
6. Klik **Simpan**
7. Informasikan password sementara tersebut kepada pengguna

## Tips Keamanan

- Segera ganti password sementara setelah berhasil login
- Gunakan password yang kuat: minimal 8 karakter, kombinasi huruf dan angka
- Jangan bagikan password kepada siapapun

## Akun Terkunci

Jika akun terkunci karena terlalu banyak percobaan login gagal (3x), tunggu **1 menit** lalu coba login kembali. Jika masih tidak bisa, hubungi Administrator IT.
