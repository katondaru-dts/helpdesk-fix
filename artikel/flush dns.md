### **a. Cara Flush DNS**  
Flush DNS digunakan untuk menghapus cache DNS yang tersimpan di komputer. Cache ini kadang bikin konflik, misalnya saat website tidak bisa diakses padahal jaringan normal.  

**Penyebab biasanya:**  
- Cache DNS sudah lama dan tidak diperbarui  
- Perubahan alamat IP website tidak terbaca  
- Error saat membuka situs tertentu  

**Solusi:**  
1. Buka CMD sebagai administrator  
2. Ketik:  ipconfig /flushdns
3. Tekan Enter dan tunggu notifikasi berhasil  
4. Coba akses ulang website
